# IPR/TPR Modeling for Intelligent Oil Wells

This repository presents computational modeling studies related to intelligent oil well systems, focusing on:

- IPR/TPR analysis
- Multiphase flow
- Productivity index estimation
- Hydraulic control systems
- Dynamic behavior
- Digital Twin concepts

## Topics

- Computational Modeling
- Applied Physics
- Fluid Mechanics
- Scientific Computing
- Numerical Simulation
- Dynamic Systems

## Tools

- Python
- MATLAB
- NumPy
- Matplotlib

## Current Goals

Develop computational tools capable of simulating intelligent multizone well behavior under different operational conditions.

## Status

Research project under development at Universidade Federal Fluminense (UFF).
